# 项目简介

## 核心功能
- 用户登录注册
- 热卖商品展示
- 优惠资讯浏览
- 投诉建议提交
- 商品分类浏览
- 商品详情查看
- 点赞功能
- 加入购物车
- 立即购买
- 商品评论
- 用户管理
- 热卖商品管理
- 订单管理
- 系统管理

## 适用场景
- 网上购物平台
- 企业内部销售系统
- 个人项目展示

## 技术特点
- 使用 Django 框架构建
- 基于 MySQL 数据库存储
- 采用 Vue 前端技术实现交互
- 代码稳定运行，测试无 bug
